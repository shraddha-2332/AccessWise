⚡ QUICK START (3 STEPS)

Step 1: Get API Key
  → Go to https://ai.google.dev/
  → Click "Get API Key"
  → Copy your free key

Step 2: Backend Setup
  → cd backend
  → npm install
  → Create .env file with: GEMINI_API_KEY=your_key_here
  → npm run dev
  Expected: "🚀 BiasAudit Backend running on http://localhost:5000"

Step 3: Frontend Setup (New Terminal)
  → cd frontend  
  → npm install
  → npm run dev
  Expected: Link to http://localhost:5173

Open browser: http://localhost:5173 🎉

   • Learn about bias types

❓ QUICK TROUBLESHOOTING

"Backend won't start"
→ Check Node.js: node --version (should be v18+)
→ Check .env has GEMINI_API_KEY
→ Port 5000 might be in use

"Frontend shows blank"
→ Check backend is running
→ Open browser console (F12) for errors
→ Clear browser cache

"API errors"
→ Verify API key is correct
→ Check internet connection
→ Text might be > 5000 characters
